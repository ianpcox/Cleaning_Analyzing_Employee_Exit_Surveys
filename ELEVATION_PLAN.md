# Project Elevation Plan: Cleaning and Analyzing Employee Exit Surveys

**Theme:** HR Analytics

## 1. Vision & Goal
Build an 'Exit Analytics for HR' product that securely processes anonymized data and uses logistic regression to predict dissatisfaction.

This plan aligns with the portfolio-wide goal of elevating projects from basic tutorials to Senior/Principal-level data science products characterized by rigorous experiment design, research-grade reporting, and software engineering best practices.

## 2. Elevation Backlog

The following phases outline the concrete steps required to elevate this project. They should be executed sequentially.

[ ] **Phase 1: Formalize HR Questions** - Define hypotheses around tenure, age, and stated reasons for leaving.
[ ] **Phase 2: Inferential Stats** - Add odds ratios or simple logistic regression to model 'dissatisfaction'.
[ ] **Phase 3: Pipeline Refactoring** - Convert cleaning steps into a robust, reproducible ETL script.
[ ] **Phase 4: HR Report** - Draft a paper-style report with actionable HR recommendations and bias/limitation notes.
[ ] **Phase 5: HR Dashboard** - Create an anonymized trend dashboard with retention alerts.
